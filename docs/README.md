# Plan today. Play tomorrow.

A developer-grade focus workspace. Task management stripped to its productive essence, paired with a Pomodoro-style deep-work engine, gamified streaks, and zero configuration.

---

## What is this.

Most productivity tools are built for neurotypical workflows — rigid lists, overflowing sidebars, feature creep disguised as value. This is not that.

Plan today. Play tomorrow. is a single-page workspace engineered for the ADHD-adjacent mind. It replaces overwhelming lists with a daily queue, replaces guilt with XP, and replaces busywork with focused deep-work blocks. No accounts. No sync. No noise.

## Key philosophy.

**Queue over list.** Every day, the system auto-picks your top tasks by priority and due date. You don't choose what to work on — you pick from what matters most.

**XP over guilt.** Complete a task, earn experience. Level up. Build a streak. The dopamine loop is intentional — productivity shouldn't feel like punishment.

**Deep work > busy work.** The focus timer auto-links to a task. Start a session, lock in, track minutes against what actually moves the needle. A completed 25-minute block on one high-priority task is worth more than checking ten boxes on busywork.

**Zero configuration.** Open the page. Start working. Everything lives in your browser's localStorage. Export if you want. Reset if you need. No sign-up, no onboarding, no database.

## Built with.

- **Astro** — static-first, zero-JS-by-default, island architecture
- **Tailwind CSS v4** — utility-first, CSS-native theming with cascading layers
- **Web Audio API** — synthesized notification tones, zero external assets
- **localStorage** — client-only persistence, no backend, no telemetry

Every dependency is intentional. Nothing is included because "everyone uses it."

## Feature set.

| Capability | Description |
|---|---|
| **Task queue** | Daily auto-pick of high-priority work. Toggle between queue view and full list. |
| **Focus timer** | Link any task, choose duration, track deep work. Timer logs accrue to the task. |
| **XP & levels** | Earn XP per task completed (low=10, med=25, high=50). Level up every 100 XP. |
| **Streaks** | Consecutive daily task completions. Fire emoji at 3+ days. Best streak tracked. |
| **Quick-add** | ⌘⇧A from anywhere. Type, Enter, done. |
| **Zen mode** | Full-screen focus overlay. Esc to exit. |
| **One-task lock** | Lock onto a single task. Everything else fades. |
| **Command palette** | ⌘K. Search, create, toggle — keyboard-first. |
| **Dark mode** | System-aware with manual toggle and localStorage persistence. |
| **Import / export** | Full JSON backup. Take your data anywhere. |
